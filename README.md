# Multithreading Cafe Simulation

This project is a Java-based cafe simulation that demonstrates multithreading and synchronisation.

It models different parts of a cafe environment using multiple threads, showing how shared resources can be managed safely.

## Features

- Multithreaded simulation
- Use of semaphores for resource control
- Simulated customer activity
- Shared resource management
- Console-based output

## Technologies Used

- Java
- Threads
- Semaphores
- Object-Oriented Programming

## What I Learned

This project helped me improve my understanding of multithreading, thread coordination, and synchronisation in Java. It also helped me understand how semaphores can be used to control access to limited resources.

## How to Run

1. Open the project in your Java IDE
2. Build the project
3. Run the main class
4. View the simulation output in the console
